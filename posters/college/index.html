Nick Cassleman - Posters - College posters

I practiced aikido throughout college and we would occasionally host demonstrations. My teacher (author of Aiki Waza) asked me to make various posters for them.

I worked for UChicago Web Services and we helped organize a data visualization contest for studies. I made this poster to help spread the word.

I took a circus performance class through UChicago's theater department. We had a final performance that we were encouraged to invite friends to, so I made this little poster for fun. Each act had a vignette of some ordinary routine (brushing teeth, cooking a meal) made extraordinary via fancy circus moves.
