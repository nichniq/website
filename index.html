Site map

My site map lists every page on this website (that I want you to know about).

I am a webpage! You can look in my insides! I left comments explaining what the code does and might even have a few secrets hidden here and there ;)

  • Right click / option click around the page until the menu says "View Page Source" and then select that
  • For safety, your browser won't open links to View Page Source, but you can copy + paste the link manually
  • Type "view-source:" at the beginning of any website URL you're on, e.g. "view-source:https://www.nichniq.name"

Another way to see how the website is made is to look at the code hosted by GitHub. It includes all of the changes over time, back from when this site was just a wee nubbin.
