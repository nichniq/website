Nick Cassleman - Seasonal playlists

I started using Spotify in 2015. They have a feature called Discover Weekly that gives you a playlist of songs they think you'll like. I started saving my favorites and round that each season collected about 20 songs, so I broke them up into seasonal playlists. Music evokes strong memory in me, so each of these playlists is very tied to time and place.

I've provided the song list, a link to the playlist on Spotify, and a button that allows you to embed a playlist widget provided by Spotify. Note that Spotify requires DRM compliance and posts a Spotify Widget Terms of Use.
